# 범쌤 사랑연구소 — 상담 상세페이지 기획안

> 파일명(예정): `coaching.html` · 홈에서 "코칭 알아보기/상담 안내"로 진입
> 성격: **세일즈 랜딩페이지** (방문자 → 신청/결제 전환)
> 주력: **1개월 밀착 코칭(40만원)** 강조 · 1시간/3개월은 비교용
> 목표(2개 병행): ① 카카오톡 무료 상담 신청  ② 바로 코칭 결제(litt.ly)

---

## 0. 전환 설계 (이 페이지가 하는 일)

```
호명("당신 얘기야") → 공감(아픔 인정) → 실패 진단("그래서 안 됐던 거예요")
→ 통찰("재회는 판단입니다") → 솔루션(1개월 코칭이 답인 이유)
→ 증거(후기·범쌤) → 반론 해소(FAQ) → 신청/결제
```

- **메인 CTA(반복):** 💬 카카오톡 무료 상담 (부담 0 → 진입장벽 제거)
- **서브 CTA(반복):** 1개월 밀착 코칭 바로 신청 (litt.ly 결제)
- CTA 노출 위치: 히어로 / 상품표 / 마지막 클로징 (최소 3회)

---

## 1. 히어로 (첫 화면 = 3초 안에 잡기)

- **헤드라인(후보):**
  - A. "읽씹, 그리고 차단. 그래도 다시 시작할 수 있습니다."
  - B. "매달릴수록 멀어졌다면, 방법이 틀린 겁니다."
  - C. "재회는 매달림이 아니라 — **판단**을 바꾸는 일입니다."
- **서브:** 이별 후 1달, 차갑게 돌아선 그 사람. 감정이 아니라 본질부터 다시 설계합니다.
- **CTA(듀얼):** [💬 무료 카톡 상담]  [1개월 밀착 코칭 신청]
- **신뢰 수치:** 누적 상담 3,000건 · 팔로워 1.8만 · 3년차
- (배경: 범쌤 프로필 또는 차분한 네이비 그라데이션)

## 2. 공감존 — "지금, 이런 상태 아닌가요?"
체크리스트(감정 자극):
- 갑자기 차가워지더니 연락이 끊기고 차단당했다
- 매달릴수록 더 멀어지는 게 느껴진다
- 무슨 말을 해야 할지, 뭘 하면 안 되는지 모르겠다
- 일이 손에 안 잡히고 밤마다 불안하다
> 마무리 한 줄: "혼자 버틴다고 나아지지 않습니다. 방향이 필요할 뿐이에요."

## 3. 실패 진단 — "이미 다 해보셨죠"
| 시도 | 왜 안 됐나 |
|------|-----------|
| 장문의 진심 메시지 | 이별의 **본질**이 아닌 곳에 초점 → 마음 안 움직임 |
| 직접 찾아가기 | 거리감은 발걸음으로 좁혀지지 않음 |
| "맞춰볼게, 고칠게" | 재회는 **판단**의 문제 — 다짐만으론 "앞으로 좋겠다"는 판단을 못 만듦 |

## 4. 통찰/철학 — "재회는 판단입니다"
- 범쌤의 핵심 관점 선언 (브랜드 철학)
- 상대는 '감정'이 아니라 '이 사람과 다시 만나면 괜찮을까'를 **판단**한다
- 그래서 매달림·읍소가 아니라, **판단을 바꾸는 본질 작업**이 필요하다
- ※ 여기에 **범쌤의 스토리(계기)** 한 단락 들어가면 신뢰 급상승 → *확인 필요*

## 5. 솔루션 — "1개월 밀착 코칭은 이렇게 진행됩니다"
진행 흐름(과정 시각화):
1. **무료 상담**으로 현재 상황 진단
2. 신청 → **사랑수업**으로 본질·판단 이해
3. **카톡 밀착 피드백**으로 실전 대화 코칭
4. **전화 전략 콜**로 결정적 순간 설계
5. **1개월 재회 복구 로드맵**으로 끝까지 동행
> "혼자 헤매지 않게, 매일 옆에서 방향을 잡아드립니다."
> ※ 실제 진행 방식(도구: 카톡/전화, 시간대, 응답 속도 등) → *확인 필요*

## 6. 상품 비교표 (Offer) — 1개월 강조
| | 1시간 코칭 | ⭐ 1개월 밀착 코칭 | 3개월 밀착 코칭 |
|---|---|---|---|
| 가격 | 10만원 | **40만원** | 130만원 |
| 사랑수업 | – | 2회 | 6회 |
| 카톡 피드백 | – | 주 3회 | 주 5회 |
| 긴급 피드백 | – | 주 1회 | 월 2회 |
| 전화 전략콜 | – | 15분×2 | 15분×6 |
| 로드맵 | 7일 플랜 | 1개월 복구 로드맵 | 90일 + 재회 후 유지전략 |
| 추천 | 급할 때 | **가장 인기 · 추천** | 확실하게 끝까지 |

- 1개월 카드에 "가장 인기" 뱃지 + 강조 테두리
- 각 카드 CTA: [이 코칭 신청하기] (litt.ly)
- 표 아래: "어떤 게 맞을지 모르겠다면? → 💬 무료 상담으로 물어보세요"

## 7. Before → After
| Before | After |
|--------|-------|
| 막막하게 고민만 반복 | 내 연애 문제의 본질을 안다 |
| 매달릴수록 멀어짐 | 어떻게 대화할지 안다 |
| 뭘 해야 할지 모름 | 감정표현 방법을 안다 |
| 불안에 무너진 일상 | 매달림 대신 나를 성장시킨다 |

## 8. 후기 (Proof)
- 실제 카카오톡 후기 이미지(16장) — 흐르는 마퀴 + "후기 전체 보기"
- 한 줄 카피: "수많은 분들이 범쌤과 함께 다시 시작했습니다"

## 9. 범쌤 소개 (Authority)
- 프로필 사진 + "재회는 판단입니다."
- 3년 / 누적 3,000건 / 팔로워 1.8만 / 본질·구체적 방법 강조
- SNS 링크(인스타·유튜브·카페·쓰레드)

## 10. FAQ (반론 해소)
- 이미 차단당했는데 가능한가요?
- 비대면으로 진행되나요? (100% 온라인)
- 결제는 어떻게 하나요? (카드/계좌)
- 기간은 얼마나 걸리나요?
- 효과가 없으면 어떡하죠? (환불/보장 정책 → *확인 필요*)
- 상대가 이미 새 사람이 생겼어도 되나요?
> ※ 자주 받는 질문/걱정 추가 → *확인 필요*

## 11. 마지막 클로징 (Closing)
- 긴급성 + 진심 톤: "망설이는 지금 이 순간에도, 골든타임은 지나갑니다."
- "완벽하지 않아도 괜찮아요. 일단 무료 상담으로 현재 상황만 점검해보세요."
- **듀얼 CTA 크게:** [💬 무료 카톡 상담]  [1개월 밀착 코칭 신청]
