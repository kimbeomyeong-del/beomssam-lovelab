## 3. 타깃 & 메시지

### 핵심 고객 (1개월 밀착 코칭 기준)
- 20~30대 여성 80% / 회사원·학생
- **이별한 지 1달 이내, 재회를 원함**
- 상대가 **차갑고 단호하게 차단·연락 두절** → 어떻게 접근할지 모름
- 일상생활이 안 될 만큼 마음이 지치고 힘든 상태

### 고객이 이미 시도했지만 실패한 것 (→ 사이트에서 "왜 안 됐는지" 짚어주기)
1. 장문 메시지 → 이별의 본질이 아닌 곳에 초점 → 효과 없음
2. 찾아가기 → 동일한 이유로 실패
3. 진심 전하기("고칠게, 맞춰볼게") → 재회는 *판단*의 문제. 맞춰주겠다는 말로는 "앞으로 좋겠다"는 판단을 못 만듦

### Before → After (대표 변화)
- Before: 막막하게 고민만, 매달림, 일상 붕괴
- After: 내 연애 문제의 본질을 앎 / 어떻게 대화할지 앎 / 감정표현 방법을 앎 / 재회에 매달리는 대신 **나를 성장**시켜야 함을 깨달음

### 차별점 (3가지 메시지로)
1. **본질을 다룬다** — 연락 스킬이 아니라 이별의 근본 원인부터
2. **구체적 방법을 준다** — "어떻게 대화하고, 어떻게 감정표현할지"까지 실전 코칭
3. **증명된 경험** — 3년차 / 누적 3,000건 상담 / 1.8만 팔로워

## 4. 전환 동선 (홈페이지의 목적)

```
방문 → (공감) 내 얘기네 → (신뢰) 이 사람 진짜네 → (행동) 무료 카톡 상담
                                                  ↘ 코칭 결제 / 강의 결제
```

- **1차 CTA(가장 강조):** 카카오톡 무료 상담 신청
- **2차 CTA:** 1개월 밀착 코칭 결제 (주력 상품)
- **3차 CTA:** 재회 긴급처방전 강의 결제 (3만원, 기획중 → "출시 알림" 처리 가능)

## 5. 페이지 구조 (정보 설계)

원페이지 스크롤 + 상단 메뉴 앵커 구조 (loveconnect와 동일 방식)

1. **헤더** — 로고 / 메뉴(소개·코칭·강의·후기·칼럼) / [카톡상담] 버튼
2. **히어로** — 대표 사진 + 핵심 카피 + CTA + 실적 수치
3. **공감존** — "이런 상황인가요?" (차단·연락두절·매달림 체크리스트)
4. **왜 안 됐나** — 장문/찾아가기/진심전하기가 실패하는 이유
5. **솔루션/철학** — 범쌤은 "본질"을 다룬다 (차별점 3)
6. **코칭 상품** — 1시간 / 1개월(주력·강조) / 3개월 가격표
7. **강의** — 재회 긴급처방전 (3만원, 곧 출시)
8. **Before→After / 후기** — 변화 사례 + 상담 후기
9. **대표 소개** — 범쌤 프로필 사진 + 경력/실적 + SNS 링크
10. **상담 신청 / 결제 CTA** — 카톡 버튼 + 신청 폼
11. **푸터** — 채널 링크(인스타·유튜브·카페·쓰레드), 사업자정보

## 6. 섹션별 카피 초안

### 히어로
- 메인: **"매달리지 않아도 되는 재회"**
- 서브: 이별의 본질을 바꾸면, 관계는 다시 시작됩니다. / 3,000건의 상담으로 증명한, 본질부터 다른 재회 코칭.
- 버튼: [무료 카톡 상담] [코칭 알아보기]
- 수치: 누적상담 3,000건 · 팔로워 1.8만 · 3년차

### 공감존 ("혹시 지금 이렇지 않나요?")
- 상대가 갑자기 차가워지고 연락이 끊겼다
- 차단당해서 어떻게 다가가야 할지 모르겠다
- 매달릴수록 더 멀어지는 것 같다
- 일이 손에 안 잡힐 만큼 마음이 무너졌다

### 왜 실패하나
- "장문 메시지, 찾아가기, '맞춰볼게'라는 말… 다 해봤지만 소용없었죠.
  재회는 **판단의 문제**입니다. 본질을 놓치면, 무엇을 해도 멀어집니다."

### 코칭 상품표
| 상품 | 가격 | 핵심 구성 |
|------|------|----------|
| 1시간 코칭 | 10만원 | 현재 상황 분석 + 하면 안 되는 행동 + 방향 + 7일 행동 플랜 |
| **1개월 밀착 코칭 ⭐** | **40만원** | 사랑수업 2회 + 카톡 피드백 주3회 + 긴급 피드백 주1회 + 전화 전략콜 15분×2 + 1개월 재회 복구 로드맵 |
| 3개월 밀착 코칭 | 130만원 | 사랑수업 6회 + 카톡 피드백 주5회 + 긴급 피드백 월2회 + 전화 전략콜 15분×6 + 90일 로드맵 + 재회 대화 준비 + 재회 후 유지 전략 |

### 강의
